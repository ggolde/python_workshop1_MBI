# Introduction
This is the homepage repository for the Python for Data Analysis: QuickStart workshop. Download the repository folder and follow the setup section.

# Setup
(MacOS only) Install homebrew, optional but recommended ([link](https://brew.sh))
1. Install VS Code ([link](https://code.visualstudio.com/download))
	- (MacOS only) With brew, run the command in your terminal: ```bash brew install --cask visual-studio-code```
2. Install miniconda ([link](https://www.anaconda.com/docs/getting-started/miniconda/install))
	- (MacOS only) With brew, run the command: ```brew install --cask miniconda```
3. Install Python and Jupyter Notebook extension collections ([tutorial](https://code.visualstudio.com/docs/getstarted/extensions))
